# Hive Merge MVP

모바일 세로 화면에 맞춘 육각형 드래그 머지 게임 MVP입니다. 보드는 처음부터 아이템으로 가득 차 있지 않고, 빈 육각 바닥 위에 화수분으로 아이템을 꺼내고 자유롭게 옮겨 같은 단계끼리 머지하는 구조입니다.

## 실행

`index.html`을 브라우저에서 바로 열면 플레이할 수 있습니다.

## 현재 특징

- 빈 칸 중심의 육각형 머지 보드
- 아이템 드래그 앤 드롭 이동
- 같은 단계 아이템끼리 머지
- 보드 안 화수분에서 랜덤 생성
- 상단에 브라운 캐릭터가 돌아다니는 연출
- 외부 오디오 파일 없이 Web Audio 합성 효과음
- 머지 순간 빔, 코어 플래시, 링, 파티클 이펙트

## 검증

```powershell
.\scripts\check.ps1
```

최소 검증으로 `node --check game.js`와 `index.html`의 에셋 참조 확인을 수행합니다.
